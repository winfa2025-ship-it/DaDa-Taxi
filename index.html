📜 私隱政策

1. 資料收集

DaDa Taxi 承諾保護您的個人私隱。我們收集的資料包括：

  • 帳戶資料（電郵、電話號碼）
  • 位置資料（用於叫車服務）
  • 付款資料（透過 Stripe 安全處理）
  • 行程記錄

2. 資料使用

我們使用您的資料提供以下服務：

  • 叫車和司機配對
  • 行程安全和追蹤
  • 客戶服務
  • 付款處理

3. 資料共享

我們可能與以下方共享您的資料：

  • Uber（用於司機配對和服務）
  • Stripe（付款處理）
  • 執法機構（法律要求時）

4. 資料安全

我們採用業界標準的安全措施保護您的資料，包括 SSL 加密和安全的伺服器存儲。

5. 您的權利

您有權：

  • 訪問您的個人資料
  • 更正不準確的資料
  • 刪除您的帳戶
  • 撤回同意

6. 聯絡我們

如有任何私隱相關問題，請透過 support@dadataxi.com 聯絡我們。
